# Continuum - Frontend Coding Assessment

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

You don't need any requisites to run this project.
If you want to run gulp task in your laptop, you need to install the package manager for Node.js packages. You can see all information [here](https://www.npmjs.com/)

### Installing

* Run on a terminal `git clone https://github.com/dianaigr/continuum.git`
* Open `index.html` of each exercise folder in your browser

### Run gulp

If you want to run the gulp task in your laptop, please follow the steps

* Run `npm install` to install Node.js dependencies.
* Run `gulp` to run the default task (or a specific NPM script) to rebuild distributed CSS and JavaScript files
* Open `index.html` in your browser to see the result
